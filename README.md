# Sudoku - Text-Based Game

This is a basic text-based Sudoku game developed in Python. Players can interact with the game through the console, filling in numbers to solve the Sudoku puzzle. The game validates moves and ensures the rules of Sudoku are followed.

---

## Features

- Play Sudoku entirely in the console  
- Validates player inputs according to Sudoku rules  
- Easy-to-run Python implementation  
- Simple and minimalistic design  

---

## Requirements

- Python 3.x  

---

## Installation and Setup

1. Clone the repository:

```bash
git clone https://github.com/davidwwdc/sudoku.git
cd sudoku
```

2. Run the game:

```bash
python main.py
```

> Replace `main.py` with the actual entry file name if different.

---

## How to Play

- The game displays a Sudoku grid in the console  
- Fill in numbers in empty cells following standard Sudoku rules  
- The game checks for valid moves and prevents invalid entries  
- Complete the grid to win
